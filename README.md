# Master's Thesis - Understanding the Adoption of Government-Backed Online Job Platforms for Repatriation: The Role of Innovation Attributes, Trust and Information Transparency
My Python code for my Master's Thesis "Understanding the Adoption of Government-Backed Online Job Platforms for Repatriation: The Role of Innovation Attributes, Trust and Information Transparency (Master of Science in Digital Marketing and Data Science, Emlyon Business school)

My repository contains two files: 
1) Thesis_Analysis.ipynb: Main data analysis notebook (e.g., Data exploration, Reliability Tests, Exploratory Factor Analysis, OLS Regression, Mediation Analysis, Data visualizations)
2) Thesis_Survey_Data.xlsx: Dataset based on sample data through an online survey
